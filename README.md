# SETTINGS-PANEL

---

A modern Settings Panel built using HTML and CSS.  
The goal of this project was to practice custom checkbox styling, structured layout grouping, and hover interactions.  
I focused on creating a clean dark-themed UI with smooth transitions and interactive elements.

---

## What this project includes

- Structured settings sections (Audio, Gameplay, Display, Accessibility)  
- Custom styled checkboxes (default browser style removed)  
- Hover effects on card and checkboxes  
- Dark themed modern UI  
- Smooth transitions and shadow effects  

---

## What I Learned

### Custom Checkbox Styling
Used `appearance: none` to remove default browser styling.  
Used `:checked` and `::after` to create a custom checkmark.  
Learned how transitions improve user interaction feel.

### UI & Interaction Design
Organizing content into logical groups improves readability.  
Hover effects enhance visual feedback.  
Box-shadow and contrast create depth in design.
